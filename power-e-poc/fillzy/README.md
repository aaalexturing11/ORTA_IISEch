# Fillzy
Sistema para reservar slots de recarga en estaciones de servicio — consulta disponibilidad, reserva y gestiona turnos.

## Descripción
**Fillzy** es una aplicación para consultar disponibilidad en estaciones de servicio y reservar **slots** para recarga.  
Ideal para reducir filas, optimizar tiempos y gestionar la demanda en puntos de carga/abastecimiento.

## Características
- Consulta en tiempo real de disponibilidad por estación.
- Reserva de slot por hora/minuto con confirmación.
- Gestión de reservas: creación, modificación y cancelación.
- Notificaciones (email / push) para recordatorios y cambios.
- Panel de administración para estaciones (horarios, capacidad, precios).
- API REST para integración con apps móviles o sistemas externos.

## Tecnologías (por determinar)
- **Backend:** Node.js / Express (o el stack que elijas).
- **Base de datos:** PostgreSQL / MySQL / MongoDB.
- **Frontend:** React / Vue / Svelte.
- **Autenticación:** JWT / OAuth.
- **Despliegue:** Docker, Kubernetes opcional.
